# 🏪 个人应用商店使用指南

## 🎯 功能概述

您现在拥有一个完整的个人应用生态系统！可以：
- 保存任何形状/智能体为可复用的应用
- 自定义应用图标和名称
- 导入/导出应用配置
- 在 Dock 栏快速访问

## 📖 使用教程

### 1️⃣ 保存应用到 Dock

**步骤：**
1. 在画布上创建一个形状（例如：AI Agent、Browser、Code Runner）
2. 配置好它的属性（例如：AI Agent 的任务描述）
3. **选中**这个形状
4. 点击 Dock 栏右侧的 **🏪 App Store** 按钮
5. 在弹出的面板中：
   - 输入应用名称（例如："翻译智能体"）
   - 选择图标 emoji（例如：🌐）
   - 点击 **Add to Dock**

✅ 现在这个应用就出现在 Dock 栏里了！

### 2️⃣ 使用自定义应用

**一键创建：**
- 点击 Dock 栏中的自定义应用图标
- 系统会在画布中心创建一个新实例
- 所有配置都已预设好！

**识别自定义应用：**
- 自定义应用右上角有一个 **绿色小点** 🟢
- 内置应用没有这个标记

### 3️⃣ 管理应用

**删除应用：**
- 鼠标悬停在自定义应用上
- 会出现一个红色的 **×** 按钮
- 点击即可从 Dock 移除

**查看所有应用：**
- 打开 App Store 面板
- 在 "📱 My Apps" 区域查看所有自定义应用

### 4️⃣ 导入/导出应用

**导出应用配置：**
1. 打开 App Store
2. 点击 **📤 Export** 按钮
3. 下载 `my-apps.json` 文件
4. 可以分享给其他人或备份

**导入应用配置：**
1. 打开 App Store
2. 点击 **📥 Import** 按钮
3. 选择 `.json` 文件
4. 应用会自动添加到 Dock

## 🎨 最佳实践

### 命名建议
- ✅ 使用清晰的描述性名称："翻译智能体"
- ✅ 包含用途："Markdown → JSON 转换器"
- ❌ 避免模糊名称："Agent 1"

### 图标选择
- 🌐 翻译/国际化
- 📊 数据分析
- 🔍 搜索/审查
- 💡 创意/头脑风暴
- 🎨 设计/美化
- 📝 写作/编辑
- 🔧 工具/修复

### 组织技巧
1. **按功能分类**：翻译类、分析类、创作类
2. **按频率排序**：常用的放在左边
3. **定期清理**：删除不再使用的应用
4. **备份配置**：定期导出保存

## 🚀 高级用法

### 创建应用模板库
```
1. 创建一系列常用的 Agent 配置
2. 导出为 "template-library.json"
3. 新项目开始时直接导入
```

### 版本管理
```
1. 导出应用时添加日期：my-apps-2026-01-17.json
2. 保留多个版本
3. 需要时回滚到旧版本
```

### 分享最佳实践
```
1. 在团队内建立应用库
2. 每个人贡献自己的高效工具
3. 定期更新和优化
```

## 📊 数据存储

- **位置**：浏览器 localStorage
- **持久性**：跨会话保持
- **容量**：通常 5-10MB
- **清除**：清除浏览器数据会删除

## 🔒 隐私与安全

- ✅ 所有数据存储在本地
- ✅ 不会上传到服务器
- ✅ 导出的 JSON 文件可以自由分享
- ⚠️ 注意：导出的配置可能包含敏感信息（如 API 密钥）
