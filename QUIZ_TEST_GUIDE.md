## ✅ 组件已创建并注册

选择题组件已经添加到项目中！

## 🎯 如何测试

### 方法 1：通过 AI 创建

1. 点击右上角的 **✨** 按钮
2. 输入：
```
创建一个选择题，问题是"JavaScript 是什么类型的语言？"，
选项是：["编译型", "解释型", "汇编语言", "机器语言"]，
正确答案是第2个
```

3. AI 会生成类似这样的 JSON：
```json
{
  "action": "create",
  "type": "quiz",
  "props": {
    "question": "JavaScript 是什么类型的语言？",
    "options": ["编译型", "解释型", "汇编语言", "机器语言"],
    "correctAnswer": 1
  }
}
```

### 方法 2：在浏览器控制台手动创建

1. 打开浏览器控制台（F12）
2. 粘贴以下代码：

```javascript
// 获取 editor 实例（Tldraw 会暴露到 window）
const editor = window.tldrawEditor || 
               document.querySelector('[data-testid="canvas"]')?.__tldrawEditor;

if (editor) {
  const center = editor.getViewportPageBounds().center;
  editor.createShape({
    type: 'quiz',
    x: center.x - 200,
    y: center.y - 160,
    props: {
      question: '1 + 1 = ?',
      options: ['1', '2', '3', '4'],
      correctAnswer: 1  // 索引从 0 开始，1 表示第二个选项 "2"
    }
  });
  console.log('✅ 选择题已创建！');
} else {
  console.error('❌ 找不到 editor 实例');
}
```

## 📝 组件功能说明

### 当前功能
- ✅ 显示题目和 4 个选项
- ✅ 点击选项提交答案
- ✅ 自动判断对错
- ✅ 显示正确答案
- ✅ 支持重试

### 交互流程
1. 用户看到题目和选项
2. 点击一个选项
3. 组件显示对错反馈
4. 如果错误，显示正确答案
5. 点击"🔄 重试"可以重新答题

## 🔧 自定义题目

修改 `props` 来创建不同的题目：

```javascript
{
  question: '你的问题？',
  options: ['选项 A', '选项 B', '选项 C', '选项 D'],
  correctAnswer: 0  // 0=A, 1=B, 2=C, 3=D
}
```

### 选项 2：预定义智能体模板

**问题**：现在 AI 生成的智能体不可控，每次任务描述都不一样

**解决方案**：创建预定义的智能体类型

```javascript
// 不再让 AI 随意生成任务
// 而是提供固定的模板

const AGENT_TEMPLATES = {
  translator: {
    name: '翻译器',
    task: '将输入的中文翻译成英文',
    icon: '🌐',
    color: '#3b82f6'
  },
  summarizer: {
    name: '摘要生成器',
    task: '提取文本的核心要点，生成简洁摘要',
    icon: '📝',
    color: '#8b5cf6'
  },
  imageAnalyzer: {
    name: '图片分析器',
    task: '描述图片内容，识别物体和场景',
    icon: '🖼️',
    color: '#ec4899'
  }
};
```

**使用方式**：
- 点击"创建翻译器"按钮 → 自动创建带固定任务的智能体
- 不需要每次都让 AI 理解你的意图

### 选项 3：相机模拟器组件

**目标**：创建一个交互式相机参数调节器

**功能**：
```
┌─────────────────────────────┐
│  📷 相机模拟器               │
├─────────────────────────────┤
│  光圈: f/2.8  [滑块]        │
│  快门: 1/125  [滑块]        │
│  ISO: 400     [滑块]        │
├─────────────────────────────┤
│  [实时预览图片]              │
│  (根据参数调整亮度/景深)     │
├─────────────────────────────┤
│  💡 AI 建议：                │
│  "光圈太小，建议调到 f/2.8" │
└─────────────────────────────┘
```

**技术实现**：
- 使用 `<input type="range">` 创建滑块
- 根据参数动态调整预览图的 CSS filter
- 将参数发送给 AI 获取建议
