## 💡 使用方法

### 场景 1：创建新 Shape（无连接）

```
步骤 1：拖出 AI Terminal
┌─────────────────────────┐
│ 💬 AI Terminal          │
│ 0 个连接                │
│                         │
│ [输入框]                │
│ "创建一个时钟"          │
│                         │
│ [▶️ 运行]               │
└─────────────────────────┘

步骤 2：输入指令
"创建一个时钟"

步骤 3：点击运行（或按 ⌘+Enter）

步骤 4：AI 自动创建新 Shape
┌─────────────────────────┐     ┌─────────────────────────┐
│ 💬 AI Terminal          │ --> │ 🕐 时钟 Shape           │
│ 1 个连接                │     │ 14:30:25                │
└─────────────────────────┘     └─────────────────────────┘
                                      ↑
                                自动创建并连接！
```

```

│ 1 个连接                │           │ 14:30:25                │
```

## 🐛 故障排除

### 问题 1：生成失败

**可能原因：**
- API Key 未配置
- 网络问题

**解决方案：**
- 检查 `.env` 文件
- 检查网络连接

### 问题 2：Shape 未创建

**可能原因：**
- AI 生成的代码有错误

**解决方案：**
- 使用更清晰的指令
- 查看浏览器控制台错误

### 问题 3：箭头连接失败

**可能原因：**
- 箭头未正确连接到 Shape

**解决方案：**
- 确保箭头两端都绑定到 Shape
- 重新拖拽箭头
